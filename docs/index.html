Features

Everything you need for edge caching

Production-ready caching infrastructure with enterprise features built-in.

⚡

Two-Tier Caching

Memory-first LRU cache with disk spillover. Hot objects served from RAM, warm objects from SSD.

📝

Write-Back Queue

Async uploads to S3 with automatic retries. Writes return immediately, synced in background.

🔗

Multipart Uploads

Full support for large file uploads with part tracking, assembly, and resume capabilities.

🌐

Distributed Coordination

Federator service coordinates cache nodes with real-time invalidation and object routing.

🔒

AWS SigV4 Auth

Native AWS Signature V4 authentication. Works seamlessly with existing S3 clients and SDKs.

📊

Prometheus Metrics

Built-in observability with cache hit rates, latency histograms, and connection metrics.

Architecture

Hub-Spoke Distribution

Central federator coordinates cache and storage nodes for consistent, low-latency access.
